# Monsters of the River

A web project focused on showcasing some of the most fascinating and fearsome river fish from around the world.  
This site presents images, brief descriptions, and information about different species, built using **HTML**, **CSS**, and **JavaScript**.

---

## Description

**Monsters of the River** is an educational and visual project that introduces several freshwater fish species known for their unique characteristics or large size.  
The website is designed to be simple, responsive, and visually attractive, with a clean structure for easy navigation.

Main goals:
- Present information about each fish in an engaging and organized way.  
- Practice responsive web design principles.  
- Implement a simple and clear layout using modern web standards.

---

## Technologies Used

- **HTML5** – Structure of the web pages.  
- **CSS** – Layout, design, and responsive styles.  
- **JavaScript** – Used for small interactive features, such as the navigation menu.  

---

## Features

- Responsive design for desktop, tablet, and mobile screens.  
- Image gallery of river fish species.  
- Clean, modern layout with flexible sections.  
- Footer credits and navigation header.
